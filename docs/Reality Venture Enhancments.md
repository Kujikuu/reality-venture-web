## **1\. The 5-Stage Project Workflow**

The core of the discussion revolves around defining a clear, sequential path for applicants through the system.

| Stage | Name | Description |
| :---- | :---- | :---- |
| **Stage 1** | **Initial / Basic** | Initial data registration. |
| **Stage 2** | **Start-up / Applying** | Filling out detailed project data. Users should receive a bilingual confirmation email (Arabic/English). |
| **Stage 3** | **Evaluation & Interview** | Setting up a 10-minute meeting (online or in-person). The admin uses a checklist to evaluate domains, logos, pitch decks, and teams. |
| **Stage 4** | **Decision** | The system marks the project as **Approved**, **Rejected**, **Suspended**, or **In Progress**. |
| **Stage 5** | **Demo Day** | Inviting the applicant to present for 5 minutes to investors. |

## ---

**2\. Technical & UI/UX Requirements**

Several specific features were requested to improve the system's professionalism and efficiency:

* **Bilingual Communication:** All automated emails and pop-up messages (e.g., "Request received successfully") must be available in both **Arabic and English**.

* **Reference ID System:** To prevent users from having to re-enter their names and phone numbers, the system will use a **Reference ID** to pull existing data automatically for subsequent stages.

* **Admin Dashboard:** Ahmed confirmed that Reference IDs are already visible in the admin dashboard, though they need to be synced with the Excel exports.

* **Social Media Integration:** The platform will prioritize **LinkedIn and X (formerly Twitter)**, as these are considered essential "identity" platforms in Saudi Arabia, even for high-ranking officials.

## ---

**3\. RV Club Enhancements**

The "RV Club" (Reality Venture Club) is being transitioned from a simple newsletter signup to a more comprehensive membership platform.

* **Additional Data Fields:** The registration will now ask for the member’s position, interests, city, and whether they work in the public or private sector.

* **Branding:** The "Stay in the loop" phrasing is being replaced with "Join the Reality Venture Club," featuring the company's specific icon and identity.
